# HET Control Integrated with Diffusion Model 
This repository contains the early-stage control architecture for a Hall Effect Thruster (HET) using both classical PID control and Stochastic Model Predictive Control (SMPC) methodologies. The project is designed to interface with a diffusion-based digital twin model and a DAQ. The long-term goal is to integrate our controllers with the HallThruster.jl simulation framework to provide physics informed state estimation and predictive modeling capabilities. The overall goal of this work is to develop a more intelligent and autonomous control system for Hall Effect Thrusters to dynamically change operating conditions.
